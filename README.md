### Основные параметры дерева

#### Поддерживаемые типы данных

| Тип данных | Описание                              | Тип C++              |
|------------|---------------------------------------|----------------------|
| **Ключ**   | нуль-терминированная строка произвольной длины | `std::string` (char*) |
| **Значение** | беззнаковое целое число             | `unsigned int`       |

#### Тип дерева и содержимое узла

**Тип дерева:** AVL-дерево (самобалансирующееся бинарное дерево поиска)

**Узел дерева должен содержать:**

```cpp
struct AVLNode {
    std::string key;                    // ключ (строка)
    std::list<unsigned int> values;     // список значений (для дубликатов)
    AVLNode* pLeft;                     // указатель на левое поддерево
    AVLNode* pRight;                    // указатель на правое поддерево
    int height;                         // высота узла (для балансировки)
};
